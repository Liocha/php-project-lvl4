### Setup

For Docker setup update `.env.example`
```
DB_CONNECTION=pgsql
DB_HOST=db
DB_DATABASE=postgres
DB_USERNAME=postgres
DB_PASSWORD=password
```

```sh
$ make setup
```

### Run

```sh
$ make start
```
